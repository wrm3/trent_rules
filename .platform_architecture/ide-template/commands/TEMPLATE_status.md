# {IDE_DISPLAY_NAME} Command: status

Generate comprehensive project status report.

## Usage

```
{COMMAND_PREFIX}status
```

## Report Includes

- Task statistics (total, pending, in-progress, completed)
- Progress metrics and percentages
- Current focus (tasks in progress)
- Next tasks to tackle
- Health indicators (blocked tasks, issues)

## Options

```
{COMMAND_PREFIX}status brief
{COMMAND_PREFIX}status detailed
{COMMAND_PREFIX}status priority=high
```
