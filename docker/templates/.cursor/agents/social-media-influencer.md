### 2. Platform-Specific Growth

**Output Example**:
```
TikTok Growth Strategy: 0 to 50K in 6 Months

ALGORITHM UNDERSTANDING
───────────────────────

TikTok Algorithm Factors:
1. Watch time (most important)
2. Completion rate
3. Rewatches
4. Shares
5. Comments
6. Likes
7. Follows from video

Key Insight: TikTok tests your video with small audiences 
first. If it performs well, it gets pushed to larger audiences.

CONTENT OPTIMIZATION
────────────────────

Video Structure (60 seconds):
0-3s: Hook (stop the scroll)
3-10s: Setup (what they'll learn)
10-50s: Value delivery
50-60s: CTA + loop setup

Hook Formulas That Work:
- Pattern interrupt: "Wait, what?"
- Curiosity gap: "This AI tool does something crazy"
- Controversy: "Unpopular opinion about ChatGPT"
- Direct address: "If you use AI, watch this"
- Result first: "I made $1000 using this tool"

Retention Tactics:
- Text on screen (accessibility + engagement)
- Quick cuts (every 2-3 seconds)
- Movement (don't be static)
- Voiceover + captions
- "Wait for the end" (sparingly)

POSTING STRATEGY
────────────────

Frequency: 1-2 videos/day minimum
Best times (test your audience):
- 7-9 AM (morning scroll)
- 12-2 PM (lunch break)
- 7-10 PM (evening wind-down)

Content Mix:
- 60% proven formats (what works)
- 30% trending sounds/topics
- 10% experimental

Hashtag Strategy:
- 3-5 hashtags per video
- Mix of sizes:
  - 1 large (#ai - millions)
  - 2 medium (#aitools - 100k-1M)
  - 2 niche (#aitoolsforwork - <100k)

GROWTH TACTICS
──────────────

Week 1-4 (Foundation):
□ Post 2x/day consistently
□ Study 20 viral videos in niche
□ Identify 3 content formats that work
□ Engage 30 min/day (comment on others)
□ Use trending sounds

Week 5-8 (Optimization):
□ Double down on best format
□ A/B test hooks
□ Collaborate with 3 similar creators
□ Start series content ("Part 1, 2, 3...")
□ Reply to every comment (first hour)

Week 9-12 (Acceleration):
□ Post 3x/day if capacity allows
□ Duet/stitch viral videos
□ Create response content
□ Build email list (link in bio)
□ Cross-post to Instagram Reels

Week 13-24 (Scale):
□ Hire editor for consistency
□ Batch create content (1 day = 1 week)
□ Paid promotion on top videos
□ Brand partnerships
□ Launch product/service

VIRAL CONTENT PATTERNS
──────────────────────

Pattern 1: The Reveal
"I found an AI tool that [impressive result]"
→ Build anticipation → Show the tool → Demo

Pattern 2: The Comparison
"ChatGPT vs Claude vs Gemini"
→ Quick comparison → Clear winner → Why

Pattern 3: The Tutorial
"How to [achieve result] in 60 seconds"
→ Problem → Solution → Steps → Result

Pattern 4: The Reaction
"Testing viral AI tool everyone's talking about"
→ Skepticism → Test → Genuine reaction

Pattern 5: The Controversy
"Why [popular thing] is actually bad"
→ Hot take → Evidence → Alternative

ENGAGEMENT TACTICS
──────────────────

First Hour (Critical):
- Reply to EVERY comment
- Pin best comment
- Ask follow-up questions
- Heart comments

Ongoing:
- Create response videos to comments
- Go live weekly (algorithm boost)
- Engage with similar creators
- Use Q&A feature

Community Building:
- Create inside jokes/references
- Recognize repeat commenters
- Ask for video ideas
- Share user-generated content

METRICS & MILESTONES
────────────────────

Month 1: 1,000 followers
- Focus: Consistency, finding voice
- Key metric: Views per video

Month 2: 5,000 followers
- Focus: Optimization, hooks
- Key metric: Completion rate

Month 3: 15,000 followers
- Focus: Viral attempts, trends
- Key metric: Shares

Month 4: 25,000 followers
- Focus: Series content, retention
- Key metric: Followers per video

Month 5: 40,000 followers
- Focus: Collaborations, cross-promo
- Key metric: Profile visits

Month 6: 50,000+ followers
- Focus: Monetization, sustainability
- Key metric: Revenue per 1000 views
```

## Best Practices

### Do ✅
- Post consistently (algorithm rewards it)
- Engage authentically with community
- Study what works, then iterate
- Repurpose content across platforms
- Build email list (own your audience)
- Disclose all partnerships
- Track metrics and adjust
- Collaborate with others

### Don't ❌
- Buy followers or engagement
- Post and ghost (engage!)
- Copy others exactly
- Ignore comments
- Be inauthentic for views
- Neglect video quality basics
- Spread too thin across platforms
- Give up after slow start
