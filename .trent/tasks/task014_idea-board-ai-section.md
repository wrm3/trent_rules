# Task 014: Update IDEA_BOARD.md template with AI-generated ideas section

## Objective
Update `template_v2/.trent/IDEA_BOARD.md` template to add a dedicated `## AI-Generated Ideas` section where autonomous agents can deposit improvement suggestions without human prompting, separate from the human-initiated `## Active Ideas` section.

## Context
The current IDEA_BOARD only captures human-initiated ideas (triggered by phrases like "make a note of that"). Autonomous cleanup agents and sprint agents may identify improvement opportunities during their runs — blocked patterns, repeated failures, bottlenecks — that would be valuable to surface. Instead of losing these insights, they should be able to write to IDEA_BOARD.

## Template Addition

Add the following section to the existing IDEA_BOARD.md template after `## Active Ideas`:

```markdown
---

## AI-Generated Ideas
<!-- Autonomous agents write here during cleanup/sprint runs -->
<!-- Human reviews these during planning sessions -->
<!-- These are SUGGESTIONS, not tasks — human decides whether to promote -->

### AI-IDEA-001: {Title}
**Status**: raw
**Category**: system-improvement | workflow | architecture | tooling | performance
**Generated**: {YYYY-MM-DD HH:MM UTC}
**Generated By**: {agent_id} during {cleanup | sprint | task_execution}
**Trigger**: {What the agent observed that prompted this idea}

**Observation**:
[What the agent noticed — specific, factual, evidence-based]

**Proposed Improvement**:
[The specific improvement suggested]

**Evidence**:
[Why the agent thinks this is worth considering — data, patterns, failures observed]

**Confidence**: low | medium | high
**Effort Estimate**: {quick-win | hours | days | major}

---
```

## AI Writing Rules (to add to rule files)

Agents MAY add to `## AI-Generated Ideas` when:
1. The same error/failure pattern appears 3+ times in cleanup scan
2. A task's `failure_history` contains `repeat_failure: true`
3. A subsystem health score drops below 60 for 2+ consecutive cleanups
4. A `spec_freshness` expiry is detected on 3+ tasks in a subsystem
5. The agent identifies a missing template or rule that would have prevented a failure

Agents MUST:
- Use the `AI-IDEA-NNN` ID format (separate from human `IDEA-NNN`)
- Include specific evidence — no vague suggestions
- Set `confidence:` honestly
- Never self-promote an AI idea to a task (human decision only)
